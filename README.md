## Project Overview
This project performs **handwritten digit recognition** using **Logistic Regression**. It uses the **digits dataset** from Scikit-learn, which contains 8x8 pixel grayscale images of handwritten digits (0–9).  

The project workflow includes:
1. **Data preprocessing** – normalize pixel values and split into training and test sets.
2. **Model training** – train a Logistic Regression model to learn patterns in handwritten digits.
3. **Prediction and evaluation** – predict labels on the test set and evaluate performance with a **classification report** and **confusion matrix**.
4. **Visualization** – display random test images with predicted and actual labels.

## Technologies and Libraries
